# @bundle-analyzer/cli

Official CLI compatible with [Bundle Analyzer service](https://www.bundle-analyzer.com).

## Install

```
npm install --save-dev @bundle-analyzer/cli
```

## Usage

```js
webpack --json | bundle-analyzer --token <your-token>
```
